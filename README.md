# Intrusion Detection System (IDS) with Machine Learning  
Projet d'apprentissage automatique — ESILV 2025

Ce projet consiste à développer un système de détection d'intrusions (IDS) basé sur des
modèles de Machine Learning, en utilisant le dataset UNSW-NB15, une référence 
dans la recherche en cybersécurité.

# Contenu du repository

- `IDS_ML.ipynb` → Le notebook complet du projet  
- `UNSW_NB15.csv` → Le dataset propre utilisé pour l’entraînement  
- `README.md` → Ce fichier

# Modèles Entraînés

Modèles classiques testés :

- Logistic Regression (binaire & multinomial)  
- Decision Tree  
- Random Forest  
- KNN  
- Softmax Regression  
- Modèle fusionné (expérimental)

GridSearchCV a été utilisé pour optimiser certains hyperparamètres.

# Résultats

Les évaluations ont été faites via :

- Accuracy  
- Precision / Recall  
- F1-Score  
- Confusion Matrix  
- Visualisation PCA binaire & multiclass
